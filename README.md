# QA Portfolio – Open Real Estate Demo App

## Overview
This repository contains an independent manual QA assessment of a publicly accessible demo real estate web application.

The goal of this project was to evaluate core user flows, identify defects, and demonstrate structured QA documentation.

## Scope Tested
- User Registration
- Login
- Property Search
- Search Filters
- Search Results
- Sorting
- Property Detail Navigation

## Testing Types Applied
- Functional Testing
- Exploratory Testing
- Negative Testing
- Regression Thinking
- Risk-Based Testing

## Deliverables Included

### QA Report
`QA_Test_Report_Open_Real_Estate.pdf`

Contains:
- Test Plan
- Scope
- Coverage Summary
- Defect Severity Breakdown
- Final Recommendations

### QA Workbook
`QA_Test_Cases_Bug_Reports_Open_Real_Estate.xlsx`

Contains:
- Test Cases
- Test Scenarios
- Bug Reports
- Execution Notes

## Skills Demonstrated
- Manual Testing
- Bug Reporting
- Severity Prioritization
- User Flow Validation
- Test Documentation
- QA Communication

## Note
This project was completed independently for portfolio purposes using a public demo environment.
